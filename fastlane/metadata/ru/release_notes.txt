Два мощных новых переводческих движка присоединяются к TLingo — Apple Translate для конфиденциальности на устройстве и Google Translate для широкого охвата языков.

Новое
• Интеграция Apple Translate — мощный движок перевода прямо на устройстве. Начните переводить мгновенно без загрузки языковых пакетов.
• Интеграция Google Translate — доступ к сервису перевода Google для ещё более широкой языковой поддержки.
• Выбор языка расширен до всех 21 языка Apple Translate.
• Определение исходного языка теперь показывает, какой язык был распознан, с помощью наглядного индикатора.
• Просматривайте установленные языковые пакеты Apple Translate прямо на странице Моделей.

Улучшено
• Страница Моделей теперь показывает только загруженные языки для более чистого отображения.
• Карточки истории переводов теперь чётко разделяют исходный текст и результат перевода.
• Боковая панель macOS переработана с использованием нативного NavigationSplitView для более плавного взаимодействия.
• Записи истории отображаются прямо в боковой панели macOS для быстрого доступа.

Исправлено
• Устранены проблемы с расположением меню переключателя языков.
• Окно загрузки языка теперь отображается в правильной позиции.
• Синхронизация настроек расширения перевода работает надёжно между приложением и расширением.
• Различные улучшения стабильности по всему приложению.
