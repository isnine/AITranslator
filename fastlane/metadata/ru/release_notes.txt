Переводите прямо на устройстве с Apple Translate — без загрузок, мгновенные результаты из приложения или системного расширения перевода.

Новое
• Интеграция Apple Translate добавляет мощный движок перевода прямо на ваше устройство. Начните переводить мгновенно — без загрузки языковых пакетов.
• Определение исходного языка теперь показывает, какой язык был распознан, с помощью наглядного индикатора, чтобы вы всегда знали, что переводится.
• Проверьте, какие языковые пакеты Apple Translate установлены на вашем устройстве, прямо на странице Моделей.

Улучшено
• Карточки истории переводов теперь чётко разделяют исходный текст и результат перевода — удобнее просматривать и находить нужное.
• Боковая панель macOS переработана с использованием нативного NavigationSplitView для более плавного и отзывчивого взаимодействия.
• Записи истории отображаются прямо в боковой панели macOS для быстрого доступа.

Исправлено
• Устранены проблемы с расположением меню переключателя языков.
• Окно загрузки языка теперь отображается в правильной позиции.
• Синхронизация настроек расширения перевода работает надёжно между приложением и расширением.
• Различные улучшения стабильности по всему приложению.
