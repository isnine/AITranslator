두 가지 강력한 새 번역 엔진이 TLingo에 추가되었습니다 — 기기 내 개인정보 보호를 위한 Apple 번역과 폭넓은 언어를 지원하는 Google 번역입니다.

새로운 기능
• Apple 번역 통합 — 기기에서 직접 작동하는 강력한 번역 엔진. 언어 팩 다운로드 없이 바로 번역을 시작하세요.
• Google 번역 통합 — Google의 번역 서비스에 접근하여 더욱 폭넓은 언어를 지원합니다.
• 언어 선택기가 Apple 번역의 21개 전체 언어로 확장되었습니다.
• 소스 언어 감지 기능이 감지된 언어를 명확한 시각적 표시로 보여줍니다.
• 모델 페이지에서 설치된 Apple 번역 언어 팩을 직접 확인할 수 있습니다.

개선
• 모델 페이지가 다운로드된 언어만 표시하여 더 깔끔한 화면을 제공합니다.
• 번역 기록 카드에서 원문과 번역 결과가 명확히 구분됩니다.
• macOS 사이드바를 네이티브 NavigationSplitView로 재설계하여 더 부드러운 경험을 제공합니다.
• 기록이 macOS 사이드바에 바로 표시되어 더 빠르게 접근할 수 있습니다.

수정
• 언어 전환기 대상 메뉴 레이아웃 문제 해결.
• 언어 다운로드 시트가 올바른 위치에 표시됩니다.
• 번역 확장 프로그램 설정 동기화가 앱과 확장 프로그램 간에 안정적으로 작동합니다.
• 앱 전반의 다양한 안정성 개선.
