더 스마트하고 세련된 TLingo — 평생 구매 옵션과 지능형 언어 매칭 추가.

새로운 기능
• 평생 구매 — 한 번 결제로 Premium을 영구 잠금 해제.
• 업그레이드 화면 개편으로 플랜 비교가 한눈에.
• macOS 온보딩 가이드 — 첫 실행 시 손쉬운 사용 설정을 안내.
• 텍스트 선택 번역 팝업에 액션 선택기 추가 — 플로팅 창을 벗어나지 않고 액션 전환.

개선
• 매치 모드가 텍스트 내용에 따라 대상 언어를 자동 감지하고 전환.
• macOS 텍스트 선택 번역이 더 빠르고 안정적 — 텍스트 가져오기 타이밍 최적화, 만료된 요청 취소, 접근성 사전 확인.
• 편집 가능한 텍스트 필드에 다듬기 액션 트리거 아이콘 표시.

수정
• Google 번역에서 원본 언어와 대상 언어가 같을 때 더 이상 오류가 발생하지 않음.
• Apple 번역의 언어 라우팅이 모든 지원 언어에서 올바르게 작동.
• Dock 아이콘 클릭 후 재실행 시 메인 창이 올바르게 표시.
• macOS에서 TestFlight 프리미엄 전환이 정상 작동.
