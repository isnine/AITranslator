Apple Translate로 기기에서 바로 번역 — 다운로드 없이 앱이나 시스템 번역 확장에서 즉시 결과를 확인하세요.

새로운 기능
• Apple Translate 통합으로 강력한 온디바이스 번역 엔진을 TLingo에 탑재. 언어 팩 다운로드 없이 바로 번역을 시작하세요.
• 소스 언어 감지 기능이 감지된 언어를 명확한 시각적 표시로 보여줘, 무엇이 번역되고 있는지 항상 알 수 있습니다.
• 모델 페이지에서 기기에 설치된 Apple Translate 언어 팩을 직접 확인할 수 있습니다.

개선
• 번역 기록 카드에서 원문과 번역 결과가 명확히 구분되어 — 더 쉽게 확인하고 참조할 수 있습니다.
• macOS 사이드바를 네이티브 NavigationSplitView로 재설계하여 더 부드럽고 반응이 빠른 경험을 제공합니다.
• 기록이 macOS 사이드바에 바로 표시되어 더 빠르게 접근할 수 있습니다.

수정
• 언어 전환기 대상 메뉴 레이아웃 문제 해결.
• 언어 다운로드 시트가 올바른 위치에 표시됩니다.
• 번역 확장 프로그램 설정 동기화가 앱과 확장 프로그램 간에 안정적으로 작동합니다.
• 앱 전반의 다양한 안정성 개선.
