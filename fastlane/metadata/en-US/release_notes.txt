Two powerful new translation engines join TLingo — Apple Translate for on-device privacy and Google Translate for broad language coverage.

New
• Apple Translate integration — a powerful on-device translation engine. Start translating immediately with no language pack downloads needed.
• Google Translate integration — access Google's translation service for even broader language support.
• Language selector expanded to all 21 Apple Translate languages.
• Source language detection now shows which language was detected with a clear visual indicator.
• View installed Apple Translate language packs directly from the Models page.

Improved
• Models page now shows only downloaded languages for a cleaner view.
• Translation history cards clearly distinguish source text from translated results.
• macOS sidebar redesigned with native NavigationSplitView for a smoother experience.
• History records display inline in the macOS sidebar for quicker access.

Fixed
• Resolved language switcher target menu layout issues.
• Language download sheet now appears at the correct position.
• Translation extension configuration syncs reliably across app and extension.
• Various stability improvements throughout the app.
