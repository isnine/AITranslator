## Description（描述）

> 限制：4000 字符

**English:**

```
TLingo is a powerful AI-powered translation app that integrates seamlessly with iOS and macOS as your default system translator.

**KEY FEATURES**

• Set as Default Translator
Replace the built-in translation with TLingo. Access AI translations anywhere through the system translation panel.

• Multiple AI Actions
- Translate: Smart translation to your target language
- Sentence Translate: Side-by-side sentence-by-sentence translation
- Grammar Check: Polish your writing with detailed error analysis
- Polish: Refine text while preserving the original language
- Sentence Analysis: Deep grammar parsing and phrase learning

• Streaming Responses
Watch translations appear in real-time with Server-Sent Events (SSE) streaming technology.

• Smart Diff Display
See exactly what changed with intuitive strikethrough and highlight formatting.

• Text-to-Speech
Listen to translations with natural AI voices.

• Native SwiftUI Experience
Beautiful, responsive interface designed for iOS and macOS.

**CUSTOMIZABLE ACTIONS**

• Create and customize your own AI actions
• Fine-tune prompts for specific use cases
• Configure multiple AI providers
• Enable/disable specific model deployments

**PRIVACY FOCUSED**

Your data stays on your device. API calls go directly to your configured providers.

**REQUIREMENTS**

• iOS 18.0+ / macOS 15.0+
• Internet connection for AI translations

Transform how you translate with TLingo – where AI meets simplicity.
```

**简体中文:**

```
TLingo 是一款强大的 AI 翻译应用，可无缝集成到 iOS 和 macOS 系统，成为您的默认翻译器。

**核心功能**

• 设为默认翻译器
替代系统内置翻译，随时随地通过系统翻译面板使用 AI 翻译。

• 多种 AI 动作
- 翻译：智能翻译到目标语言
- 逐句翻译：原文译文对照显示
- 语法检查：润色文本并分析错误
- 润色：在保持原语言的情况下优化表达
- 句子分析：深度语法解析和短语学习

• 流式响应
采用 SSE 流式技术，实时显示翻译结果。

• 智能差异对比
通过删除线和高亮直观展示修改内容。

• 文字转语音
使用自然 AI 语音朗读翻译结果。

• 原生 SwiftUI 体验
专为 iOS 和 macOS 设计的精美界面。

**自定义动作**

• 创建和自定义您自己的 AI 动作
• 针对特定场景调整提示词
• 配置多个 AI 提供商
• 启用/禁用特定模型部署

**注重隐私**

您的数据保留在设备上，API 请求直接发送到您配置的提供商。

**系统要求**

• iOS 18.0+ / macOS 15.0+
• 需要网络连接进行 AI 翻译

TLingo —— 让 AI 翻译触手可及。
```
